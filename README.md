# Convert: Tutor Assignment Tool

## Overview

Convert is a React-based web application designed to simplify the process of managing tutor assignments. It offers a user-friendly interface for inputting client details and generates formatted output for various tutor categories.

## Features

- **Form Handling**: Collects client information including name, contact, level, subjects, and more.
- **Customizable Outputs**: Generates tailored outputs for different tutoring categories.
- **Nearest MRT Calculation**: Integrates with `nearest-mrt` to find the closest MRT stations based on postal codes.
- **Dynamic Fee Calculation**: Uses a comprehensive fee structure for different levels and tutor categories.
- **Code Generation**: Unique code generation for each client based on name and level.

## Technologies Used

- React.js
- `nearest-mrt` Library
- Local Storage for Data Persistence

## Setup

Clone the repository, install dependencies, and run the application:

```bash
git clone https://github.com/kab0o0m/PremiumTutors.git
cd https://github.com/kab0o0m/PremiumTutors.git
npm install
npm start

```
